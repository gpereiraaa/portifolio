Projetos em Back-End

API de controle de filmes de uma locadora

Neste projeto, utilizei Node.js para construir uma API RESTful de uma locadora de filmes, usando Prisma para comunicação com o banco SQL. O desafio principal aqui é garantir a organização e escalabilidade do código, entre os padrões disponíveis, a arquitetura MVC foi a escolhida como proposta de aplicação. A API está 100% documentada com Swagger.

    Ferramentas Utilizadas:
  • Node.js
  • Express
  • Prisma
  • Cors
  • SQL
  • Arquitetura MVC
  • Swagger
  • GitHub

API para o Whatsaap

Neste projeto, utilizei Node.js para construir uma API que simula o WhatsApp, usando um arquivo JSON como fonte de dados. O desafio principal era percorrer todo esse arquivo criando endpoints que retornam contatos de certos usuarios, dados pessoais, palavras chaves de mensagens e outros. A API está 100% documentada com Postman e disponível online via Render.

    Ferramentas Utilizadas:
  • Node.js
  • Express
  • Cors
  • JSON
  • Postman
  • GitHub
  • Render

Projetos em Front-End

Site de personagens da Disney

Neste projeto, criei um site interativo que permite aos usuários explorar o vasto universo de personagens da Disney. Utilizando a API pública da Disney, a aplicação exibe uma galeria de personagens, permite a busca por nomes específicos e apresenta detalhes sobre cada um.

    Ferramentas Utilizadas:
  • HTML
  • CSS
  • JS
  • Disney API
  • GitHub

Site Liturgia diária

Neste projeto, criei um site interativo que permite aos usuários explorarem a liturgia do dia e também de um dia específico. Utilizando um API publica que retorna todas as orações do dia, também as leituras do dia e evangelho.

    Ferramentas Utilizadas:
  • HTML
  • CSS
  • JS
  • API Liturgia Diária
  • GitHub
